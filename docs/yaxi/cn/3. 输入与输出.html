输入与输出

输出

输出用到的函数是print，是一个内置函数（关于函数的详细说明，请见“使用函数”一章）。这个函数可接受任意多个的参数，每一个参数就是每一行中输出的内容。传入的参数值可以是任意类型，因为在输出时都会转化为str类型。使用举例：

@print("Hello World!", "Hello, Yaxi!")

输出结果：

$ Hello World!
$ Hello, Yaxi!

注意，print前的@是使用函数的标识符。

输入

输入使用的是input函数，也是一个内置函数，仅接受一个参数，这个参数用于显示输入提示。输入函数返回值类型是str，所以用来储存输入内容的变量需要是str类型的，或者str类型的可转化类型（关于返回值类型和数据类型转化请分别参考“使用函数”和“变量”这两章）。代码示例：

str name = @input("Please input your name: ")

@print("Hello " + @name)
