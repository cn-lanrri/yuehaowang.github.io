条件判断

基本用法

程序语句中。条件判断是一个非常常用的功能。使用条件判断语句，你可以实现选择性执行代码这一功能。几乎所有编程语言都有这一句法，在yaxi中，条件判断语句基本格式如下：

if condition
    // condition为true，执行这部分代码
else
    // condition为false，执行这部分代码
endif

condition是一个获取布尔值的表达式。参看一下说明可以帮助你理解什么是“获取布尔值的表达式”：

在程序设计中，我们会经常遇到需要使用“是与不是”的情形，“是”在程序中叫做true，“不是”叫做false。遇到这种情况，我们会用到获取“是与不是”的一个表达式，这个表达式将会返回一个布尔型数据。最简单的一个例子是比较两个数的大小，我们可以将其简化为“是与不是”问题，假设我们想比较2与7的大小，表达式2 < 7就将得到true，也就是2 < 7这个命题成立。再例如true && true、false || true这些都是获取布尔值的表达式，当然直接使用布尔型变量来获取布尔值也是可以的。

在条件判断的应用场景中，一个非常经典的例子就是用于判断考试分数是否及格，以下代码可以实现这一过程：

// 假设分数是75
int grade = 75

if @grade < 60 // 不及格情况
    @print("及格")
else // 及格情况 
    @print("不及格")
endif

由于分数是75，所以输出的内容将是“及格”。

有的时候，我们不需要else里的内容，就可以直接这么写：

if condition
    // 执行所需代码
endif

条件判断语句必须使用endif来标记结束，否则代码无法正常运行。

拓展用法

有的时候条件判断会迎来多条件判断的挑战。使用基本的语法会略显麻烦，以下的用法或许可以帮你脱离困境。

if condition_A
    // 条件A满足，执行这里的代码
elseif condition_B
    // 条件B满足，执行这里的代码
elseif condition_C
    // 条件C满足，执行这里的代码
else
    // 条件A、B、C都不满足，执行这里的代码
endif

值得一提的是，条件判断语句执行时，会按顺序检测条件（从上到下）， 一旦遇到条件表达式返回值是true的就直接运行该条件对应的代码，剩下的条件和代码都会被跳过。
