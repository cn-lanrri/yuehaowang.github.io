Yaxi 中文文档

变量

数据类型

yaxi提供了以下几种数据类型：

  1. int 整型。可以是[-2147483648, 2147483647]内的任意整数。
  2. float 浮点数。用于表示小数。
  3. bool 布尔值。只能是true或false。
  4. str 字符串。用于储存一个字符串。

定义变量

在yaxi中定义变量请使用以下的代码：

int a = 1
float b = 1.0
str c = "Yaxi"

值得注意的是，定义变量的时候不能使用关键字作为变量名，yaxi中的所有关键字请在“概况”一章查看。除了这一限制，变量名也不能与其他变量以及函数重名（关于函数请见“使用函数”一章）。

另外还需注意的是，定义字符串的时候，字符串内容要在一对单引号'或引号"之间。但是如果使用str s1 = "You are "a man"!!!"或者str s1 = 'You are 'a man'!!!'这样的方式定义字符串，就会引起麻烦，因为定义字符串时，双引号之间不能再有双引号，单引号之间也不能再有单引号。解决方案分为两类：1, 将双引号内的双引号改用单引号，单引号内的单引号改用双引号；2, 使用\转意符号，使用如下：str s1 = "You are \"a man\"!!!"

当然，定义变量的时候，你也可以使用数学表达式，例如：

int a = 1 + 2

提示 关于变量的操作，后文会有提及。

当然你也可以在变量定义的时候不进行赋值操作，那么变量将以默认值进行赋值：

int a  // 0
str s  // ""
bool b  // false
float f  // 0.0

使用变量

在使用变量时需要用到@标识符：

str s = "Hello, Yaxi"

@print(@s)

改动变量时，同样如此：

int a = 1

@a = 3

变量的操作

变量的操作主要会用到数学表达式以及逻辑表达式，表达式是一个可以返回数据的式子，比如1 + 2返回3，2 < 3返回true。不同数据类型的操作存在差异，以下就是几种数据类型常用操作的介绍。

整型和浮点数的数学运算

整型和浮点数的运算除了支持+ - * /操作外，还有可以使用%进行取余数运算，参考下面的代码：

int a = 9 + 13 // result: 22
int b = 1 / 3 // result: 0
float c = 1 / 3    // result: 0.333333
int d = 23 % 2 // result: 1

字符串的操作

字符串操作相对比较单一，只有加法。而加法则是直接将两个字符串相拼接。如下：

str s1 = "1" + "2" // result: "12"
str s2 = "Hello " + "yaxi" // result: "Hello yaxi"

布尔型变量的操作

布尔型变量主要用于逻辑操作，支持以下运算符号：

  • ! 类似于数学中命题的否定，即!true = false，!false = true
  • || 类似于数学中的“或”，即true || true = true，true || false = true，false || false = false
  • && 类似于数学中的“且”，即true && true = true，true && false = false，false && false = false

数据类型的转换

不通数据类型之间可以进行一定的转换，也就是说，字符串可以转化为整数，反过来，整数也可以转化为字符串。当然数据转化也不仅仅只有上述两种。以下是几个种转换的示例：

// 数字转字符串
str s3 = 1 + 4 // result: "5"
// 布尔型转字符串
str s4 = false // result: "false"
// 字符串转整数
int n1 = "1" + "4" // result: 14
// 字符串转布尔型
bool b1 = "true" // result: true
// 字符串转浮点数
float f1 = "1" + "0.1" // result: 10.1

值得注意的是，有时候数据类型转化是非法的。比如将字符串"yaxi"转为int类型。由于这类的字符串不代表任何数字，所以在转化的过程中就会遇到麻烦。同理，上述类似的字符串也无法转化为bool类型。但一般而言，无论变量类型、变量值是多少，都可以转化为str类型。
