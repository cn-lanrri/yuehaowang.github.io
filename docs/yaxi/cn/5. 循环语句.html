循环语句

基本语法

循环语句同样作为非常常用的功能在程序设计中不可或缺。Yaxi中循环语句句法如下：

loop condition
    // 需要重复执行的代码
endloop

其中condition同条件语句中的condition含义相同，是一个获取布尔值的表达式，当condition为true时就会重复执行循环语句中的代码直到condition变为false。

以下是一个非常基础的循环案例 —— 从一加到十：

int i = 1

int result

loop @i <= 10
    @result = @result + @i

    @i = @i + 1
endloop

请注意，通常在循环语句中必须要保证某次循环后能做到退出循环，也就是condition变为false，否则循环无法退出，会出现程序崩溃的问题，这就是所说的“死循环”。

中断循环

在使用循环语句的过程中，有时会遇到需要中途中断循环的情况。这时候可以使用break语句。示例如下：

int i

loop true
    if @i == 10
        break
    endif

    @i = @i + 1
endloop
