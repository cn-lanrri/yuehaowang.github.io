Yaxi 中文文档

概况

简介

Yaxi /jəsi/ 是一款基于C++和Qt的简易脚本语言。Yaxi为你提供了定义变量，定义函数，循环，条件判断等句法，你可以通过yaxi十分轻松地写出有趣的小程序。同时，使用yaxi，你还可以进行简单算法的研究，让你熟悉程序设计的过程，感受程序设计的乐趣。

当然，yaxi作为一款简易的脚本语言，缺乏对数据结构，类等高级功能的实现，所以使用yaxi进行工程项目的开发是不太现实的。不过，yaxi因提供了简单实用的语法，所以作为计算机编程新人的入门语言是十分合适的。

最初印象

这是一段yaxi代码：

int a = 727

// 输出变量a
@print(@a)

上述代码可以看到，yaxi中的每一行代码就是一条条的程序指令，多条程序指令不能放在同一行。同时，yaxi与C++，Java等语言不同的地方在于，每一条程序命令后不能使用 ;作为分隔符。程序设计的过程中，往往会使用到注释，用于对代码进行注解且不会影响程序的正常运行，yaxi中使用//进行注释，可以在每行代码之后，也可以独占一行。

首行代码的作用是定义一个int类型的变量，变量名称为a，并将其赋值为727，这一句法和C语言中的定义变量的语句十分相似。print是yaxi语言的一个内置函数，用于标准输出，@符号是使用变量和函数的标识符。运行这段代码，得到的结果将是在屏幕上输出“727”。

关键字

Yaxi中有一些特定的字符串不能作为变量和函数的名称，否则会导致使用时出现异常错误。以下是关键字列表：

| int       | str       | bool      |
| --------- | --------- | --------- |
| void      | float     | true      |
| --------- | --------- | --------- |
| false     | if        | endif     |
| --------- | --------- | --------- |
| elseif    | else      | loop      |
| --------- | --------- | --------- |
| endloop   | func      | endfunc   |
| --------- | --------- | --------- |
| return    | print     | input     |
| --------- | --------- | --------- |
| null      | goto      |           |
