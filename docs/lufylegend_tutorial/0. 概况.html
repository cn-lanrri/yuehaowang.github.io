简介

引擎介绍

lufylegend是一个HTML5开源引擎,它实现了利用仿ActionScript3.0的语法进行HTML5的开发， 包含了LSprite，LBitmapData，LBitmap，LLoader，LURLLoader，LTextField，LEvent等多个AS开发人员熟悉的类， 支持Google Chrome，Firefox，Opera，IE9，IOS，Android等多种热门环境。 利用lufylegend可以轻松的使用面向对象编程，并且可以配合Box2dWeb制作物理游戏， 另外它还内置了LTweenLite缓动类等非常实用的功能, 现在开始使用它吧，它可以让你更快的进入HTML5的世界！

lufylegend目前仅支持Canvas渲染，不过在未来会推出WebGL渲染模式，敬请期待～

教程介绍

本教程是lufylegend的入门教程。无论你之前是否从事过游戏开发，本教程都会引导你一步一步地探索lufylegend，从而成为一名HTML5游戏开发者。在开始学习之前，你需要有一定的HTML和JavaScript基础，能使用HTML写出像样的网站，以及能够调试JavaScript程序并且对JavaScript的面向对象编程有所了解。关于HTML和JavaScript的教程请自行百度／Google，网上有成堆的教程。此外，本教程中统一使用最新版本的lufylegend，如果你在使用本教程中的代码进行学习时，浏览器提示报错，请检查是否使用的是最新版本引擎。

库件包介绍

首先从官方网站下载库件最新版。解开下载好的压缩包后，你将得到一个库件目录。

库件包内包含lufylegend-x.x.x.js和lufylegend-x.x.x.min.js两个完整版本，还有lufylegend-x.x.x.simple.js和lufylegend-x.x.x.simple.min.js两个缩减版本，***.min是去除了回车和空格的压缩版本。

simple缩减版本与完整版本的区别在于，将LBox2d，LQuadTree，LTransitionManager，LoadingSample1，LoadingSample2，LoadingSample3，LoadingSample4，LoadingSample5 等几个类与引擎的常用部分分离，缩减了引擎的体积。如果用到了被分离的部分的功能的话，可以进行手动加载。

除了上面几个库件.js文件，下载包含的api.html是API手册，examples/包含了一些示例，src/是引擎源代码，lib/是一些附加库件，比如上面提到的LBox2d，LQuadTree，LTransitionManager等，README.md是说明文件。
